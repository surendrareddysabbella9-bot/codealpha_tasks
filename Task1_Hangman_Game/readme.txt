# Hangman Game

## Description

This is a console-based Hangman Game developed using Python. The player must guess a hidden word one letter at a time before running out of chances.

## Features

* Multiple difficulty levels (Easy, Medium, Hard)
* Different word collections for each difficulty
* Dynamic chances based on difficulty level
* Input validation
* Duplicate guess detection
* Play Again option
* Sound effects for correct and wrong guesses

## Technologies Used

* Python
* Random Module
* Winsound Module

## How to Run

1. Open the terminal.
2. Navigate to the project folder.
3. Run the following command:

python hangman.py
