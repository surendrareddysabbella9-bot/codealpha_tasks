# Fun Chatbot

## Description

This is a rule-based chatbot developed using Python. The chatbot responds to predefined user inputs and includes fun interactive features.

## Features

* Greetings
* Random jokes
* Fun facts
* Roast mode
* Mood-based responses
* Love advice
* Date and time information
* Rock Paper Scissors mini-game
* Help menu
* Exit command

## Technologies Used

* Python
* Random Module
* Datetime Module

## How to Run

1. Open the terminal.
2. Navigate to the project folder.
3. Run the following command:

python chatbot.py

## Available Commands

* hello
* hi
* how are you
* joke
* fact
* roast me
* i am happy
* i am sad
* love advice
* date
* time
* game
* help
* bye
