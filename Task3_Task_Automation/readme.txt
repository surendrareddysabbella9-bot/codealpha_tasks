# Email Extractor

## Description

This Python script extracts email addresses from a text file and saves them into a separate output file.

## Features

* Reads data from a text file
* Extracts email addresses automatically
* Removes duplicate email addresses
* Counts total emails found
* Saves extracted emails to a new file

## Technologies Used

* Python
* Regular Expressions (re)
* File Handling

## Files

* input.txt
* emails.txt
* email_extractor.py

## How to Run

1. Add text content containing email addresses to input.txt
2. Open the terminal.
3. Run the following command:

python email_extractor.py
