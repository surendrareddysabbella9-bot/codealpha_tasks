# CodeAlpha Python Development Internship

## Overview

This repository contains the tasks completed as part of the CodeAlpha Python Development Internship. The projects were developed using Python and demonstrate fundamental programming concepts such as loops, conditional statements, file handling, dictionaries, regular expressions, and user interaction.

## Tasks Completed

### Task 1: Hangman Game

A console-based word guessing game where players attempt to guess a hidden word before running out of chances.

#### Features

* Multiple difficulty levels
* Dynamic chances based on difficulty
* Input validation
* Duplicate guess detection
* Play Again option
* Sound effects

#### Concepts Used

* Random Module
* Lists
* Loops
* Conditional Statements
* User Input Handling

---

### Task 2: Stock Portfolio Tracker

A simple stock tracking application that calculates the total value of a user's stock portfolio using predefined stock prices.

#### Features

* Displays available stocks
* Accepts multiple stock entries
* Portfolio summary generation
* Total investment calculation

#### Concepts Used

* Dictionaries
* Loops
* Conditional Statements
* Arithmetic Operations

---

### Task 3: Email Extractor

A Python script that extracts email addresses from a text file and saves them to a separate file.

#### Features

* Email extraction using Regular Expressions
* Duplicate email removal
* Email counting
* Output file generation

#### Concepts Used

* Regular Expressions
* File Handling
* Lists
* Pattern Matching

---

### Task 4: Fun Chatbot

A rule-based chatbot that responds to predefined user inputs and provides interactive features.

#### Features

* Greetings
* Jokes
* Fun Facts
* Roast Mode
* Mood-Based Responses
* Date and Time Information
* Rock Paper Scissors Mini Game

#### Concepts Used

* Conditional Statements
* Loops
* Random Module
* Datetime Module
* User Interaction

---
