# Stock Portfolio Tracker

## Description

This project calculates the total investment value of a stock portfolio using predefined stock prices and user-provided quantities.

## Features

* Displays available stocks and prices
* Accepts multiple stock entries
* Calculates investment value
* Generates portfolio summary
* Displays total investment amount

## Technologies Used

* Python
* Dictionary
* Loops
* Conditional Statements

## How to Run

1. Open the terminal.
2. Navigate to the project folder.
3. Run the following command:

python stock_tracker.py

## Learning Outcomes

* Dictionary Operations
* User Input Handling
* Arithmetic Operations
* Portfolio Calculations
* Data Storage Using Dictionaries
